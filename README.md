## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/mithunkumar3112/portfolio.git
```

Move to the project folder

```bash
cd portfolio
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build for production

```bash
npm run build
```

Preview production build

```bash
npm run preview
```
